# testrepo_spacex
Repository for IBM capstone (Data Science Certification)

In the following repository, various Data Science tools and methods are used. It begins with web scraping a table from Wikipedia, which is then transformed through EDA into a dataset to be analyzed using Python.

To accomplish this, the necessary libraries are used for each stage of the process.

After data manipulation, SQL queries are performed for data exploration, and visualizations are created using Matplotlib.

Additionally, the Folium library is used to generate coordinates and perform map-based analysis, while Plotly Dash is used to create interactive charts.
